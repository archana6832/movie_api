Movie API

This is a movie api that can be used to access a list of movies that includes Movie Title and its description, Genre, Directors and User information.

Business Logic/Request URL HTTP Method Request body data format Response body data format
Get a list of movies /movies GET None A JSON object holding data about all the movies
Get data about a single movie /movies/[moviename] GET None A JSON object holding data about a single movie including an ID, movie name and other details about the movie. Example:{
id : 1,
title: "Luca",
director: " Tom Harry"
}
Get a list of movies by genre /movies/genres/[genre] GET None A JSON object holding data of a list of movies of a particular genre
Get data about a director by name /directors/[directorName] GET None A JSON object holding data of a particular director
Add a new user /users POST A JSON object holding data about a new user to add,structure like:
{
user id : 5,
name : "Mathew Thomas",
email : mathew@gmail.com
}		 A JSON object holding data about a new user that was added,structure like:
{
user id : 5,
name : "Mathew Thomas",
email : mathew@gmail.com
}
Update user information /users/[userName] PUT None A text message indicating the user information updated
Add to favorites /users/[userName]/movies/[movieName] POST A JSON object containing the movie name. A JSON object containing the movie name.
Delete from favorites /users/[userName]/movies/[movieName] DELETE None A text message indicating the movie is removed from favourites
Delete a user /users/[userName] DELETE None A text message indicating a user is removed.

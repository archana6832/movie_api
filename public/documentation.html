Movie API

This is a movie api that can be used to access a list of movies that includes Movie Title and its description, Genre, Directors and User information.

Business Logic/Request URL HTTP Method Request body data format Response body data format
Get a list of movies /movies GET None A JSON object holding data about all the movies
Get data about a single movie by title /movies/[moviename] GET None A JSON object holding data about a single movie.
Example:

{
"Genre": {
"Name": "Genre's name",
"Description": "Description of the genre."
},
"Director": {
"Name": "Director's Name",
"Bio": "Director's biography",
"Birthyear": "YYYY",
"Deathyear": "YYYY"
},
"_id": objectId,
"Title": "Movie Name",
"Description": "Movie description",
"imageUrl": "www.link.to/movie/coverartwork",
"Featured": Boolean
}

Get data about a specific genre /movies/genres/[genre] GET None A JSON object holding data about a particular genre
Get data about a director by name /directors/[directorName] GET None A JSON object holding data of a particular director
Example:

{
"_id": objectId,
"Name": "Director's Name",
"Bio": "Director's biography",
"Birthyear": "YYYY",
"Deathyear": "YYYY"
}

Add a new user /users POST A JSON object holding data about a new user to add,structure like:

{
"Username": "Username",
"Password": "Password",
"Email": "example@mail.com",
"Birthday": "YYYY-mm-dd",
}

A JSON object holding data about a new user that was added including an ID .The structure is as follows:

{
"ID": "UserID",
"Username": "Username",
"Password": "Password",
"Email": "example@mail.com",
"Birthday": "YYYY-mm-dd",
}

Update user information /users/[userName] PUT A JSON object holding data about user info to update,
Example:

{
"Password": "newpassword" }

A JSON object holding data of a user info that is sucessfully updated,
Example:

{
"_id": objectId,
"Name": "Firstname",
"Username": "Username",
"Password": "New Password",
"Email": "email-address@mail.com",
"Birthday": "YYYY-mm-dd",
}

Add to favorites /users/[userName]/movies/[movieID] POST None A JSON object containing the user information.
Example:

{
"_id": objectId,
"Name": "Firstname",
"Username": "Username",
"Password": "Password",
"Email": "email-address@mail.com",
"Birthday": "YYYY-mm-dd",
"FavoritesMovies": [MovieID1]
}

Delete from favorites /users/[userName]/movies/[movieID] DELETE None A JSON object containing the user information.
Example:

{
"_id": objectId,
"Name": "Firstname",
"Username": "Username",
"Password": "Password",
"Email": "email-address@mail.com",
"Birthday": "YYYY-mm-dd",
"FavoritesMovies": [MovieID1]
}

Delete a user /users/[userName] DELETE None A text message indicating a user is removed.
